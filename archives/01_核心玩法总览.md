# 游戏设计文档：《代号：工匠之心》 - 核心玩法总览

## 1. 游戏核心愿景 (Core Vision)

这是一款 **1P VS 1P 战棋游戏**。其核心是 **“所玩即所造”**——玩家通过一个深度且有意义的挂机制造系统，亲手打造和自定义每一个单位的装备、卡牌和战斗风格。

游戏的胜负手不再是单纯的数值碾压，而是玩家的**战略规划、团队配合、资源管理和临场应变能力**。我们致力于创造一个公平、易于上手、但战术天花板极高的竞技环境。

---

## 2. 核心系统一：战棋对战系统 (The Battlefield)

这是游戏的“执行”层面，是策略与博弈的舞台。

### 2.1 基础规则与设定

*   **格式**: 3棋子 vs 3棋子 团队对抗。
*   **棋盘**: 在一个由六边形或方形格子组成的战术地图上进行。
*   **胜利条件**: 主要以目标为导向（如据点争夺），辅以全歼敌方为次要胜利条件，并设有回合数上限以防对局冗长。

### 2.2 棋子与属性

*   **单位 (Unit)**: 玩家操控的棋子，每个单位代表一个独特的职业（如圣骑士、游侠等）。
*   **核心属性**:
    *   **生命值 (HP)**: 单位的生存能力。
    *   **移动力 (Movement)**: 一个固定的数值（例如，坦克2，游侠4），代表该单位在激活时免费移动的最大格数。

### 2.3 回合与行动流程

*   **蛇形激活 (Snake Draft)**: 为保证公平，采用交替激活顺序。5v5的顺序示例为：`玩家A(1) -> 玩家B(2) -> 玩家A(2) -> 玩家B(2) -> 玩家A(2) -> 玩家B(1)`。一个大轮回合中，双方各行动相应数量的单位次。
*   **单位激活流程 (A Unit's Turn)**:
    1.  **激活**: 玩家选择一个本轮尚未行动的单位进行“点亮”。
    2.  **资源与抽牌**: 该单位从自己的牌库抽3张牌。
    3.  **免费移动**: 玩家可以免费移动该单位，移动格数不超过其【移动力】属性。此操作不消耗任何资源。
    4.  **卡牌行动**: 移动后，玩家消耗该单位当前拥有的能量 (Energy) 来打出手中的卡牌。
    5.  **回合结束**: 未使用的手牌全部进入弃牌堆，但玩家可以选择保留1张到下一次该单位抽牌时再抽到。

### 2.4 能量系统 (Energy System)

*   **独立能量池**: 每个单位拥有独立的能量池，上限为3点。
*   **恢复机制**: 在己方大回合开始时（轮到你激活第一个单位前），你场上所有存活单位各自恢复1点能量。
*   **累积机制**: 能量会累积，未用完不清零，直至达到3点上限。

### 2.5 卡牌与装备系统：深度自定义

*   **卡牌获取**: 游戏中的所有卡牌（包括基础卡和消耗品）均由挂机系统制造，是消耗品，使用后需重新制造。
*   **模块化卡牌“炼成”**:
    *   **基础卡牌 (Base Card)**: 拥有固定卡图和符文插槽的“模板”，通过挂机制造获取。稀有度决定插槽潜能。
    *   **符文镶嵌 (Rune Inlay)**: 将挂机制造的“符文石”镶嵌在卡牌上，赋予或改变卡牌效果，并决定其最终的能量消耗。
*   **分级装备系统**:
    *   **低级装备 (普通/优秀)**: 提供简单、直接、泛用的效果（例如 `移动力+1` 或 `生命值+2`），作为新手过渡和高级装备的合成材料。
    *   **高级装备 (稀有/史诗)**:
        *   **构筑核心装备 (Archetype Gear)**: 通常在[护符]槽位。不提供直接数值，而是赋予单位一个流派“关键词”（如 `[火焰]`, `[剧毒]`），作为使用该系别符文的**“许可证”**。
        *   **战术特效装备 (Tactical Gear)**: 提供独特的被动或条件性能力，奖励特定的战术行为（例如：“若你本回合没有移动，回合结束时获得2点护盾”）。

### 2.6 平衡性机制

*   **击杀奖励**: 击杀敌方单位的棋子，在它下一次被激活时，将额外获得1点能量。
*   **劣势补偿**: 当己方有单位阵亡后，己方所有剩余单位在下一次被激活时，会额外多抽1张牌（此效果持续一整个大轮）。

## 3. 挂机系统详解 (The Workshop)

**核心定位**: 一个需要主动管理的、具有内部成长循环的**“模拟工坊”**，为战棋对决提供源源不断的、可定制的“弹药”和“零件”。

### 3.1 玩法循环

*   **派遣任务**: 派遣穿戴生产装备（矿镐、药锄等）的英雄进行采集、探索、狩猎等任务，获取多种材料。
*   **制造合成**: 通过制造技能（锻造、炼金、符文雕刻）将材料合成为基础卡牌、符文石、装备、一次性消耗品卡牌。

### 3.2 长线吸引力 (Endgame Loop)

全解锁后，挂机系统的意义从“解锁”转变为：

*   **持续生产“消耗品卡牌”**: 为 PVP 和 PVE 对战提供战术储备。
*   **制造“门票”**: 产出参与高阶玩法（如周末锦标赛、PVE“虚空远征”）所需的入场券。
*   **追求顶级装备/符文**: 制造并试验不同流派的“构筑核心装备”和极其稀有的“传说符文”。
*   **参与限时活动**: 生产活动所需的特殊材料，兑换绝版外观和奖励。

## 4. 地图系统 (The Map System)

**核心定位**: 地图是战斗的第三位参与者，其设计旨在鼓励互动、打破僵局。

### 4.1 设计要素

*   **地形**: 障碍物、掩体（森林）、复杂地形（沼泽）等将对移动和攻防产生直接影响。
*   **互动元素**: 地图上将设置战略争夺点（如占点区域）、资源点（提供临时 buff）或动态危险区域，引导玩家进行交战。
*   **占点规则**: 采用 **“绝对控制”** 规则，即据点内只有一方单位时，该方才能在自己的回合开始时得分，有效避免“人海”战术和对峙。
*   **预热阶段**: 开局第一大轮，关键目标点处于“未激活”状态，鼓励双方进行战术布局和试探，而非无脑冲锋。
